Getting Started

Installation#

You can find all available releases on GitHub, which will come bundled with most of the requirements for getting it up and running. Select and download the most recent release to date for your platform (i.e. Windows, MacOS or GNU/Linux) and run the installer.

Once installed, please follow the instructions to download and configure the UF VPN Client.

At this point, installation is complete and you may continue on to the next section.
