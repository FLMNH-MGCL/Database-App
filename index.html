SpeSQL

Software for Managing FLMNH Collection Data

Get Started

Easy to Use

SpeSQL was designed in a way to be easily installed and used by researchers unfamiliar with SQL syntax.

Customizable Queries

SpeSQL supports both basic and advanced variants for issuing all the fundamendal CRUD database operations (create, read, update and delete).

Designed for Researchers

SpeSQL was thought of by and developed for the researchers at the McGuire Center for Lepidoptera and Biodiversity, at the Florida Museum of Natural History.

Completely Open Source

SpeSQL was developed using TypeScript React and is publically hosted on a GitHub repository.
